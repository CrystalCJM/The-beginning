## 我学会的语法

### 1. 标题
用 `#` 表示标题，几个 `#` 就是几级标题。

### 2. 重点标记
用 `**文字**` 表示**粗体**，用 `*文字*` 表示*斜体*。

### 3. 列表
无序列表用 `-` 或 `*`：
- 苹果
- 香蕉
- 橙子

有序列表用数字：
1. 第一步
2. 第二步
3. 第三步

### 4. 代码
行内代码：`` `git clone` `` → `git clone`

代码块：
```bash
# 这是注释
echo "Hello, Markdown!" 

``` 

### 5. 引用
> 学习最好的方法就是动手做。——非常同意

### 6.  分割线
用---画一条线把内容分隔开

上面这条线就是分割线，单独一行写  ---  就行，不需要整数倍，3 个就够了。
